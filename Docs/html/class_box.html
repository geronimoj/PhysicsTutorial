Parameters
positionThe position of the origin of the box
isKinematicShould the box be simulated by the physics engine
velocityThe inital velocity of the boxes Center of Mass in units per second
massThe mass of the Center of Mass
colourThe colour of the box in RGBA format
extentsThe half extents of the box
orientationThe inital orientation of the box in degrees
angularVelocityThe inital angular velocity of the box in radians per second or per second
layerThe layer of the box. Determines what it can collide with. Defaults to layer 0
linearDragThe linear drag of the box. Defaults to 0
angularDragThe angular drag of the box. Defaults to 0
elasticityThe elasticity of the box. Determines how much energy is gained or lost from a collision with this physics object. Defaults to 1 (no energy loss or gain)
staticFrictionThe static friction coefficient. The friction factor applied on a stationary object. Defaults to 1. Set to 0 for frictionless
kinematicFrictionThe kinetic friction coefficient. The friction factor applied on a moving object. Defaults to 1. Set to 0 for frictionless

Determines if this box is intersecting with the given box and returns information about the contact point.

Parameters
boxThe box to compare to
contactThe point of contact. Does not override the original value, it instead adds to the value.
numContactsThe number of contacts between the boxes. Does not override the original value, it increments the value instead.
penA reference to a float for writing purposes. How far the contact point penetrates the box.
edgeNormalA reference to a glm::vec2 for writing purposes. The normal of the edge the collision occured on.
Returns
Returns true if there is a collision and writes to the parameters information about the collision.
